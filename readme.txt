REST API Accelerator
====================

REST API testing Accelerator/starter suite.


Overview
--------

This repository provides an example how to test REST API's with Copado Robotic Testing.

Examples of GET and POST operations are included and we are using two open API's
(OpenLibrary and Unix TimeStamp). 3rd party library "RequestsLibrary" is used here
for making API calls.


Note that there are other libraries available as well for
this purpose, for example:

* RESTInstance
* OpenAPIDriver

...to name a few.


Additional information
----------------------

Directory structure:

* Tests are under "tests" folder
* Supporting files are under "resources" folder
